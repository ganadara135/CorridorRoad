# Alignment, Stations, and Profile

Alignment, Stations, and Profile define the corridor control line and finished-grade vertical intent.

## Alignment

Alignment owns horizontal source intent.

Use it to define or edit the horizontal path used by downstream stationing, profile, and corridor evaluation.

## Stations

Stations create the station grid used by section application and review.

Station checks help identify:

- stations outside alignment range
- stations not covered by station rows
- boundary tolerance issues

## Profile

Profile owns finished-grade vertical intent.

Profile supports:

- PVI station/elevation rows
- CSV-style input paths where available
- Auto Interpolate Elevations for filling missing elevation rows between known start/end values
- station check review

## Review

Use Plan/Profile Connection Review to check whether Alignment, Stations, and Profile agree before building heavier downstream results.
